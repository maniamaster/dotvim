Installation:

    git clone git://github.com/maniamaster/dotvim.git ~/.vim

Create symlinks:

    ln -s ~/.vim/vimrc ~/.vimrc

Install vundle:

    git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

    :BundleINstall from vim

in .bashrc:

    TERM=xterm-256color
